# 使用与授权说明（安全增强版）

## 一、文件清单
- `video_prompt_tool.py` —— 主程序（最终打包成 exe 发给用户）
- `licensing.py` —— 授权校验模块（含**公钥**，可公开）
- `secure_store.py` —— API 密钥本地加密（Windows DPAPI）
- `updater.py` / `version.py` —— 自动更新与版本号
- `keygen.py` —— **激活码生成器，只给你自己用**（用私钥签发激活码）
- `generate_keys.py` —— **一次性**生成 Ed25519 密钥对
- `private_key.pem` —— **签名私钥，绝密！** 不上传 GitHub、不打包、不外发
- `build_exe.bat` / `VideoPromptTool.spec` —— 一键打包脚本

## 二、安全机制说明（和旧版的区别）
1. **授权：非对称签名（Ed25519）**
   - 程序里只有**公钥**，只能“验证”激活码，**无法伪造**。
   - 能签发激活码的**私钥**只在你电脑上（`private_key.pem`）。
   - 别人即使反编译 exe，也拿不到私钥，做不出能用的激活码。
   - ⚠️ 因为算法换了，**旧版 HMAC 激活码全部失效**，老用户需要你用新 keygen 重发一次。
2. **API 密钥：DPAPI 加密存储**
   - 密钥用 Windows 账户级加密保存在 `config.json`，不再是明文。
   - 换 Windows 账户 / 换电脑都无法解密，拷走配置文件也没用。
3. **启动即校验**：打开程序先进激活界面校验，通过才进主界面；
   含系统时间回拨检测（防止改系统时间绕过到期）。

## 三、首次配置（只做一次）
1. 安装依赖：`python -m pip install cryptography`
2. 运行：`python generate_keys.py`
   - 生成 `private_key.pem`（私钥，**保管好，别外发**）。
   - 屏幕会打印一行 `PUBLIC_KEY_HEX = "..."`。
3. 把那行公钥粘贴到 `licensing.py` 顶部的 `PUBLIC_KEY_HEX`（覆盖原值）。
   - keygen 用 `private_key.pem` 签发，主程序用 `licensing.py` 里的公钥验证，配对即可。
   > 本仓库已内置一对可用的密钥与公钥；如要换成你自己的，按上面三步重做即可
   > （注意：换密钥会让已发出的激活码失效）。

## 四、打包成 exe
1. 双击 `build_exe.bat`（或 `pyinstaller --noconfirm VideoPromptTool.spec`）。
2. 等出现 `SUCCESS!`，成品在 `dist\VideoPromptTool.exe`，这个 exe 发给用户。
   - 私钥 `private_key.pem` **不会**被打进 exe（程序只用公钥）。

## 五、给用户激活
1. 用户运行 exe，进入“软件激活”窗口，里面有一串**机器码**。
2. 用户把机器码发给你。
3. 你运行 `python keygen.py`（它会自动加载同目录的 `private_key.pem`），
   粘贴机器码，选到期日期（或勾“永久”），点“生成激活码”→“复制”。
4. 把激活码发给用户，用户填进激活窗口、点“激活”即可。
5. 到期后软件提示重新激活；再发一个新日期的激活码即可“续期”。

## 六、发布新版本 + 自动更新
1. 改 `version.py` 里的 `APP_VERSION`（如 `1.1.0` → `1.2.0`）。
2. 提交并打 tag 推送：
   ```
   git add -A && git commit -m "release v1.2.0"
   git tag v1.2.0
   git push && git push --tags
   ```
3. GitHub Actions 会自动打包并发布 Release。
4. 老用户打开软件时会自动检测到新版本，确认后自动下载、替换、重启。
   - 也可手动点界面上的“检查更新”。
   - 需要仓库为**公开**（自动更新免令牌）。`version.py` 里的
     `GITHUB_OWNER/GITHUB_REPO` 要填成你的仓库。

## 七、其它说明
- 设置（地址、模型、fps、提示词、预设、主题、激活码）保存在
  `用户目录\.video_prompt_tool\config.json`，下次启动自动带出。
  其中 **API 密钥为 DPAPI 加密**。
- 提示词“预设”：写好后点“另存为预设”，下次从下拉框直接选；可“恢复默认”。
- 界面右上角可一键切换**浅色 / 深色**主题。
- 安全提醒：非对称签名能挡住绝大多数伪造/破解，但任何本地校验都无法
  100% 防住专业逆向（例如直接 patch 程序跳过校验）。需要最强防护请改联网校验。
